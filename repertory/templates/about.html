How It Works

Current venues are restricted to Austin Film Society (at the Marchesa and the Screening Room), the Paramount and Stateside Theatres, and all Austin Alamo Drafthouse locations. That's it for now, although if you're interested in adding a venue and can provide a reliable data source (or update a Google Sheet) for it, let's talk.

The calendar is updated once or twice a week with anything that has tickets on sale. For the purposes of this calendar, "rep" is roughly defined as anything more than three years old.

Built on Open Source

Thanks also to all the communities of open source projects and their contributors that power this application. Python, Django, Redis, PostgreSQL, SQLite, jQuery, jQuery Mobile, Bootstrap, and several supporting libraries, as well as The Movie Database—which is not open source, but is a great, free, user-editable resource with a robust API—made it possible for version one of this app to go up after one all-night coding session, so thank you.

The code for this is all open source, btw. Pull requests welcome.

Top features on my mind
  1. There should be an easy-to-use import interface (rather than just the command-line option available now)
  2. There should be a movie modal that shows other play times, a poster or other image, running times and ticket links when possible
  3. You should be able to see which of your Facebook friends are going to an event (if they want you to)
  4. You should be able to remove items while viewing your personal calendar
